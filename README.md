# Univariate-Time-Series-Forecasting-using-ARIMA-SARIMAX

This project focuses on forecasting future values of a univariate time series using **ARIMA** and **SARIMAX** models. The goal is to analyze the dataset, build statistical forecasting models, evaluate their performance, and visualize predictions.

---

## üìå Project Overview

Time series forecasting is widely used for predicting trends in finance, sales, energy consumption, and many real-world scenarios. In this project:

- We explore and visualize the time series data
- Perform stationarity checks and differencing
- Build and tune **ARIMA** and **SARIMAX** models
- Compare forecasting accuracy
- Generate forecast plots for future time periods
